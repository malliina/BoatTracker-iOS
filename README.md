# BoatTracker-iOS

The iOS app for BoatTracker. See [www.boat-tracker.com](https://www.boat-tracker.com) for details.

## Development

To run tests from the command line:

    xcodebuild test -workspace BoatTracker.xcworkspace -scheme BoatTracker -destination 'platform=iOS Simulator,name=iPhone 13 mini,OS=17.0.1'

### Formatting

    brew install swift-format
    swift-format format --configuration swift-format.json -i -r .

## Releasing

1. Update the version number. It appears that the App Store does not allow version number reuse.
1. Every commit to the master branch triggers a production release to the [App Store](https://itunes.apple.com/us/app/boat-tracker/id1434203398?ls=1&mt=8) via [appcenter.ms](https://appcenter.ms/apps).
